Put Ready Events in the Game

Local game connected

This mirrors every event marked Canon or Ready for game into data/events. Drafts, ideas, and parking-lot events stay out of the game. The files stay only if Godot's EventCheck passes.

The game loads event files when it starts. Restart the game after installing new events.

Share Everything

This sends all saved events at once. It does not send only the event currently open.

On a phone, tap Share Everything and choose Mail, Messages, or another app. If that does not work, tap Copy Everything and paste it where you need it.

Advanced: files and backups

Use backup when moving devices or saving a full copy of the workspace. Game files are separate JSON files, one per event.
